# 💼 Personal Portfolio Website

A modern, responsive, and interactive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**. This portfolio showcases my skills, projects, resume, and contact information with smooth animations and a clean user interface.

The website is designed to provide recruiters and visitors with an overview of my work in **Web Development**, **Machine Learning**, and **Artificial Intelligence**.

```bash

```

Open the project folder

```bash

```

Run the project

Simply open **index.html** in your browser.

Or use **VS Code Live Server** for a better development experience.
